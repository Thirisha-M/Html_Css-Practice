1. Introduction

Machine learning (ML) is a subset of artificial intelligence (AI) that enables systems to learn patterns and make decisions based on historical data. It focuses on using algorithms to analyze data, learn from it, and then make a determination or prediction about something in the world.[1]

2. Types of Machine Learning

2.1 Supervised Learning

Supervised learning algorithms learn from labeled data, making it suitable for tasks like classification and regression. Common examples include spam detection and image recognition.[2]
